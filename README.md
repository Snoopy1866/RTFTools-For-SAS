# RTFTools for SAS

![Github License](https://img.shields.io/github/license/Snoopy1866/RTFTools-For-SAS)

## 简介

适用于 SAS 的 RTF 文件处理程序。

以下编码环境可用：

- [utf8](src/utf8/)
- [utf16](src/utf16/)
- [gbk](src/gbk/)
- [gb18030](src/gb18030/)
